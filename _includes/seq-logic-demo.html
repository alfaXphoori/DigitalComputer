📝 อธิบายการทำงานแบบเรียลไทม์: อินพุต A = 0 และ สถานะปัจจุบัน Q = 0 ทำให้สมการลดรูปได้ D = 0 เมื่อเกิดสัญญาณนาฬิกา สถานะจะไม่เปลี่ยนแปลง

💡 คู่มือการเรียนรู้: สังเกต Feedback Loop (สายย้อนกลับจาก Memory ไปยัง Combinational Logic) ค่าสถานะปัจจุบัน Q จะป้อนกลับไปคำนวณร่วมกับอินพุต A เพื่อกำหนดค่าสถานะถัดไป D ล่วงหน้า แต่ Q จะไม่เปลี่ยนค่าจนกว่าจะกดปุ่ม Pulse CLK (หรือเปิด Auto-Tick) เพื่อสร้างสัญญาณขอบขาขึ้น!
