## ส่วนที่ 1: ทำความเข้าใจ SR Latch (10 นาที)

### SR Latch จาก NAND Gate

```
  S̄ (Active LOW) ──┤1  ├── Q
                    │NAND│
              ┌────┤2  ├──┐
              │    └────┘  │
              │            │
              │    ┌────┐  │
              └───┤1  ├──┘
                   │NAND│
  R̄ (Active LOW) ──┤2  ├── Q̄
                    └────┘
```

> ⚠️ **NAND SR Latch ใช้ Active LOW** — กด 0 เพื่อ Set/Reset

### Truth Table (NAND SR Latch)

| S̄ | R̄ | Q | Q̄ | สถานะ |
|:---:|:---:|:---:|:---:|:---|
| 0 | 1 | **1** | 0 | **SET** — Q = 1 |
| 1 | 0 | 0 | **1** | **RESET** — Q = 0 |
| 1 | 1 | Q | Q̄ | **HOLD** — คงค่าเดิม |
| 0 | 0 | 1 | 1 | **FORBIDDEN** ❌ — ห้ามใช้ |

### IC 7400 — Quad 2-input NAND

```
        ┌───U───┐
  1A  ──┤1    14├── VCC
  1B  ──┤2    13├── 4B
  1Y  ──┤3    12├── 4A
  2A  ──┤4    11├── 4Y
  2B  ──┤5    10├── 3B
  2Y  ──┤6     9├── 3A
 GND ──┤7     8├── 3Y
        └───────┘
```

## ส่วนที่ 2: ต่อวงจร SR Latch (25 นาที)

### ขั้นตอน

1. วาง IC 7400 บน breadboard
2. ต่อ VCC (pin 14) และ GND (pin 7)
3. **NAND₁** (เกตบน): pin 1 = S̄ input, pin 2 = feedback จาก Q̄, pin 3 = Q
4. **NAND₂** (เกตล่าง): pin 4 = feedback จาก Q, pin 5 = R̄ input, pin 6 = Q̄

### การต่อ Feedback (สำคัญมาก!)

```
NAND₁ output (pin 3 = Q)  ──→ NAND₂ input (pin 4)
NAND₂ output (pin 6 = Q̄) ──→ NAND₁ input (pin 2)
```

5. ต่อ **Push Button + Pull-up 10kΩ** สำหรับ S̄ และ R̄:
   - ปกติ = HIGH (ผ่าน pull-up)
   - กดปุ่ม = LOW (Active)
6. ต่อ LED ที่ Q (pin 3) และ Q̄ (pin 6)

## ส่วนที่ 3: ทดสอบ (20 นาที)

### ทดสอบที่ 1 — SET

1. กดปุ่ม S̄ (ให้ S̄ = 0, R̄ = 1)
2. สังเกต LED Q และ LED Q̄
3. **ปล่อยปุ่ม** (S̄ กลับเป็น 1, R̄ = 1 → Hold)
4. Q ยังคงค่าอยู่หรือไม่?

| ขั้นตอน | S̄ | R̄ | LED Q | LED Q̄ | สถานะ |
|:---|:---:|:---:|:---:|:---:|:---|
| กด S̄ | 0 | 1 | | | |
| ปล่อย S̄ | 1 | 1 | | | |

### ทดสอบที่ 2 — RESET

1. กดปุ่ม R̄ (ให้ S̄ = 1, R̄ = 0)
2. สังเกต LED
3. ปล่อยปุ่ม

| ขั้นตอน | S̄ | R̄ | LED Q | LED Q̄ | สถานะ |
|:---|:---:|:---:|:---:|:---:|:---|
| กด R̄ | 1 | 0 | | | |
| ปล่อย R̄ | 1 | 1 | | | |

### ทดสอบที่ 3 — HOLD (Memory!)

1. ทำ SET ก่อน (กด S̄ แล้วปล่อย) → Q = 1
2. ไม่กดปุ่มใดเลย (S̄ = 1, R̄ = 1)
3. Q ยังเป็น 1 อยู่หรือไม่? → **นี่คือ Memory!**

### ทดสอบที่ 4 — FORBIDDEN ❌

1. **กดทั้ง S̄ และ R̄ พร้อมกัน** (S̄ = 0, R̄ = 0)
2. สังเกต: Q และ Q̄ เป็นอะไร?
3. ปล่อยทั้งสองปุ่มพร้อมกัน → Q เป็นอะไร? (ไม่แน่นอน!)

| ขั้นตอน | S̄ | R̄ | LED Q | LED Q̄ | หมายเหตุ |
|:---|:---:|:---:|:---:|:---:|:---|
| กดทั้งสอง | 0 | 0 | | | |
| ปล่อยทั้งสอง | 1 | 1 | | | ผลไม่แน่นอน? |

📸 **Screenshot สถานะ SET (Q=1, Q̄=0)**
