## ส่วนที่ 4: Parity Bit — ตรวจจับ Error (15 นาที)

### แบบฝึก 4.1 — เพิ่ม Even Parity Bit

จงเพิ่ม Parity Bit (บิตซ้ายสุด) ให้จำนวน 1 ในข้อมูลรวมเป็น **เลขคู่**

| ข้อมูล (7 บิต) | จำนวน 1 | Parity Bit | ข้อมูลพร้อม Parity (8 บิต) |
|:---:|:---:|:---:|:---:|
| 100 1011 | | | |
| 011 0100 | | | |
| 111 1111 | | | |
| 000 0000 | | | |

### แบบฝึก 4.2 — ตรวจจับ Error

ข้อมูลต่อไปนี้ใช้ **Even Parity** จงตรวจว่ามี error หรือไม่

| ข้อมูลที่ได้รับ (8 บิต) | จำนวน 1 | มี Error? |
|:---:|:---:|:---:|
| 1 100 1010 | | |
