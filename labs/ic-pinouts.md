# 🔌 ตารางตำแหน่งขาไอซี (IC Pinout Quick Reference)

เอกสารนี้รวบรวมแผนภาพตำแหน่งขา (Pinout Diagram) ของไอซีลอจิกพื้นฐานซีรีส์ 74xx และ 45xx ที่ใช้ในปฏิบัติการวิชาตรรกศาสตร์ของดิจิตอลคอมพิวเตอร์ เพื่อให้นิสิตสะดวกในการอ้างอิงขณะต่อวงจรทดลองบนบอร์ดจริงและใน Tinkercad

## 🛠️ คำแนะนำพื้นฐานในการต่อวงจรลอจิกไอซี

1. **ระบุตำแหน่งขา 1:** สังเกต "รอยบาก (Notch)" หรือ "จุดกลม (Dot)" บนตัวไอซี ให้หันรอยบากไปทางซ้าย (หรือขึ้นด้านบน) ขา 1 จะอยู่ด้านซ้ายล่างสุด จากนั้นนับวนทวนเข็มนาฬิกา
2. **ต่อสายไฟเลี้ยงก่อนเสมอ:** ทำการต่อขา **VCC** (ต่อเข้าแหล่งจ่าย +5V) และขา **GND** (ต่อลงดิน 0V) ของชิปทุกตัวให้เสร็จก่อนเริ่มเดินสายสัญญาณอื่น ๆ
3. **อย่าปล่อยขาอินพุตลอย (Floating Input):** ขาอินพุตของเกตที่ไม่ได้ใช้งาน ควรต่อลง GND หรือ VCC ผ่านตัวต้านทาน เพื่อป้องกันสัญญาณรบกวน (ยกเว้นขาสัญญาณเอาต์พุต ห้ามต่อชนไฟเลี้ยงหรือดินโดยตรง)

## 1. กลุ่มเกตตรรกะพื้นฐาน (DIP-14)

### 7404 (NOT)
ตัวแปลงสัญญาณลอจิกจำนวน 6 ประตูภายในชิปเดียว ($Y = \overline{A}$)

```text
             7404 Hex Inverter
                 ---___---
          1A  1 |*        | 14  VCC (+5V)
          1Y  2 |         | 13  6A
          2A  3 |         | 12  6Y
          2Y  4 |         | 11  5A
          3A  5 |         | 10  5Y
          3Y  6 |         |  9  4A
         GND  7 |         |  8  4Y
                 ---------
```

---

### 7408 (AND)
เกตแอนด์ 2 อินพุต จำนวน 4 ประตู ($Y = AB$)

```text
             7408 Quad 2-Input AND
                 ---___---
          1A  1 |*        | 14  VCC (+5V)
          1B  2 |         | 13  4B
          1Y  3 |         | 12  4A
          2A  4 |         | 11  4Y
          2B  5 |         | 10  3B
          2Y  6 |         |  9  3A
         GND  7 |         |  8  3Y
                 ---------
```

## 2. กลุ่มวงจรคอมบิเนชันระดับกลาง (DIP-16)

### 7483 (4-bit Adder)
วงจรบวกเลขฐานสองขนาด 4 บิตแบบขนาน พร้อมขา Carry-In ($C_0$) และ Carry-Out ($C_4$)

```text
             7483 4-Bit Full Adder
                 ---___---
          A4  1 |*        | 16  C4 (Carry Out)
          S3  2 |         | 15  B4
          A3  3 |         | 14  S4 (Sum 4)
          B3  4 |         | 13  C0 (Carry In)
         VCC  5 |         | 12  GND
          S2  6 |         | 11  B1
          B2  7 |         | 10  A1
          A2  8 |         |  9  S1 (Sum 1)
                 ---------
```

---

### 74138 (Decoder)
วงจรถอดรหัสลอจิกขนาด 3 อินพุต ออก 8 เอาต์พุต (Active-LOW)

```text
             74138 3-to-8 Decoder
                 ---___---
           A  1 |*        | 16  VCC
           B  2 |         | 15  Y0' (Active Low)
           C  3 |         | 14  Y1'
        G2A'  4 |         | 13  Y2'
        G2B'  5 |         | 12  Y3'
          G1  6 |         | 11  Y4'
         Y7'  7 |         | 10  Y5'
         GND  8 |         |  9  Y6'
                 ---------
```

* **ขาเลือกข้อมูล (Address Inputs):** A (บิตต่ำสุด LSB), B, C (บิตสูงสุด MSB)
* **ขาสั่งการทำงาน (Enable Inputs):** ต้องป้อนลอจิกให้สอดคล้องเพื่อให้ชิปทำงาน:
  * G1 = `1` (Active-HIGH)
  * G2A' = `0` (Active-LOW)
  * G2B' = `0` (Active-LOW)

---

### 74151 (Multiplexer)
วงจรเลือกรับสัญญาณข้อมูล 8 ช่องทาง ออก 1 ช่องทาง

```text
              74151 8-to-1 MUX
                 ---___---
          D3  1 |*        | 16  VCC
          D2  2 |         | 15  D4
          D1  3 |         | 14  D5
          D0  4 |         | 13  D6
           Y  5 |         | 12  D7
          Y'  6 |         | 11  A (Select S0 - LSB)
     STROBE'  7 |         | 10  B (Select S1)
         GND  8 |         |  9  C (Select S2 - MSB)
                 ---------
```

* **ขาสัญญาณข้อมูล:** D0 ถึง D7
* **ขาเลือกสัญญาณ:** C (S2), B (S1), A (S0)
* **เอาต์พุต:** Y (สัญญาณตามที่เลือก) และ Y' (สัญญาณกลับค่า)
* **ขาเปิดการทำงาน (Strobe):** ต้องต่อลง GND เพื่อให้ชิปทำงานปกติ (Active-LOW)

---

### 4511 (7-Segment Driver)
ไอซีถอดรหัสรหัส BCD (4 บิต) เพื่อขับจอแสดงผล 7 ส่วนชนิด Common-Cathode (ต่อลงดินร่วม)

```text
             4511 BCD to 7-Seg Driver
                 ---___---
           B  1 |*        | 16  VDD (+5V)
           C  2 |         | 15  f
         LT'  3 |         | 14  g
         BL'  4 |         | 13  a
         LE   5 |         | 12  b
           D  6 |         | 11  c
           A  7 |         | 10  d
         GND  8 |         |  9  e
                 ---------
```

* **ขาอินพุตข้อมูล BCD:** D (MSB), C, B, A (LSB)
* **ขาควบคุม:**
  * LT' (Lamp Test): ต้องต่อ VCC (ป้อน 1) ในโหมดทำงานปกติ (ถ้าต่อ GND จะสว่างทุกเซกเมนต์)
  * BL' (Blanking): ต้องต่อ VCC (ป้อน 1) ในโหมดทำงานปกติ (ถ้าต่อ GND จอจะดับทั้งหมด)
  * LE (Latch Enable): ต้องต่อ GND (ป้อน 0) ในโหมดทำงานปกติ เพื่อให้จอรันตามอินพุตแบบ Realtime
* **เอาต์พุต (a-g):** ขับแต่ละเซกเมนต์ของ LED 7-Segment โดยตรงผ่านตัวต้านทานกระแสไฟ

## 3. กลุ่มวงจรเชิงลำดับและเก็บข้อมูล (DIP-14 / DIP-16)

### 7473 (JK Flip-Flop)
ฟลิปฟลอปแบบ JK ทำงานที่ขอบสัญญาณนาฬิกาขาลง (Negative edge-triggered) พร้อมขา Clear เป็นอิสระ
⚠️ *หมายเหตุ: ขา VCC อยู่ที่ขา 4 และขา GND อยู่ที่ขา 11*

```text
              7473 Dual JK Flip-Flop
                 ---___---
        1CLK  1 |*        | 14  1J
        1CLR  2 |         | 13  1Q
          1K  3 |         | 12  1Q'
         VCC  4 |         | 11  GND
        2CLK  5 |         | 10  2K
        2CLR  6 |         |  9  2Q
          2J  7 |         |  8  2Q'
                 ---------
```

---

### 7474 (D Flip-Flop)
ฟลิปฟลอปแบบ D ทำงานที่ขอบสัญญาณนาฬิกาขาขึ้น (Positive edge-triggered) พร้อมขา Reset (CLR) และ Preset (PR) แบบ Active-LOW

```text
              7474 Dual D Flip-Flop
                 ---___---
        1CLR' 1 |*        | 14  VCC
          1D  2 |         | 13  2CLR'
        1CLK  3 |         | 12  2D
         1PR' 4 |         | 11  2CLK
          1Q  5 |         | 10  2PR'
         1Q'  6 |         |  9  2Q
         GND  7 |         |  8  2Q'
                 ---------
```

---

### 7490 (Counter)
ไอซีตัวนับทศนิยมแบบอซิงโครนัส (Decade Counter - MOD-10)

```text
             7490 Decade Counter
                 ---___---
        CLKB  1 |*        | 14  CLKA
         R01  2 |         | 13  NC
         R02  3 |         | 12  QA
          NC  4 |         | 11  QD
         VCC  5 |         | 10  GND
         R91  6 |         |  9  QB
         R92  7 |         |  8  QC
                 ---------
```

* **ขาสัญญาณนาฬิกาอินพุต:** CLKA และ CLKB
* **เอาต์พุต:** QA (LSB), QB, QC, QD (MSB)
* **การรีเซ็ต:**
  * R01, R02 (Reset to 0): ต้องต่อลง GND เพื่อให้วงจรทำการนับปกติ
  * R91, R92 (Set to 9): ต้องต่อลง GND เพื่อให้วงจรทำการนับปกติ
* **การต่อใช้งาน MOD-10:** นำสัญญาณนาฬิกาอินพุตป้อนเข้า CLKA และต่อขาสัญญาณเอาต์พุต QA เชื่อมต่อกับขาสัญญาณนาฬิกา CLKB จากนั้นนำผลลัพธ์การนับออกทาง QA, QB, QC, QD

### 74194 (Shift Register)
เรจิสเตอร์เลื่อนข้อมูลขนาด 4 บิตแบบเอนกประสงค์ (Universal Shift Register) สามารถหมุน ซ้าย-ขวา เก็บข้อมูลแบบขนาน หรือโฮลด์ค่าได้

```text
             74194 Universal Register
                 ---___---
         MR'  1 |*        | 16  VCC
         DSR  2 |         | 15  Q0 (Parallel Out 0)
          P0  3 |         | 14  Q1
          P1  4 |         | 13  Q2
          P2  5 |         | 12  Q3 (Parallel Out 3)
          P3  6 |         | 11  CP (Clock Input)
         DSL  7 |         | 10  S1 (Mode Control)
         GND  8 |         |  9  S0 (Mode Control)
                 ---------
```

* **โหมดควบคุม (Mode Control):**
  * S1 = 0, S0 = 0 → โหมดคงค่าเอาต์พุตเดิม (Hold)
  * S1 = 0, S0 = 1 → เลื่อนบิตไปทางขวา (Shift Right) โดยนำบิตเข้าที่ขา DSR
  * S1 = 1, S0 = 0 → เลื่อนบิตไปทางซ้าย (Shift Left) โดยนำบิตเข้าที่ขา DSL
  * S1 = 1, S0 = 1 → โหลดข้อมูลอินพุตแบบขนาน (Parallel Load) จากขา P0-P3 ทันทีเมื่อเกิดขอบนาฬิกาขาขึ้น
* **ขารีเซ็ต (MR'):** เป็นขาเคลียร์ข้อมูลแบบไม่ประสานจังหวะ (Asynchronous Clear) ต้องต่อ VCC (ป้อน 1) ในโหมดทำงานปกติ หากต่อลง GND (ป้อน 0) เอาต์พุตทั้งหมดจะกลายเป็น 0 ทันที
