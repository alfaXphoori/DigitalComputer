## ทฤษฎีย่อเฉพาะ Lab 9

Decoder เป็นวงจรเชิงผสม (combinational circuit) ที่แปลงรหัสไบนารี `n` บิตให้เป็นเอาต์พุต `2^n` เส้น
สำหรับ 3-to-8 decoder อินพุต 3 บิตคือ `A`, `B`, `C` และเอาต์พุตคือ `Y0` ถึง `Y7`
ในใบงานนี้กำหนดให้ `A` เป็นบิตมากสุด (MSB) และ `C` เป็นบิตน้อยสุด (LSB)

เมื่อ enable `EN = 1` เอาต์พุตของ decoder แบบ active-high จะทำงานเพียงเส้นเดียว เช่น
`ABC = 101` ทำให้ `Y5 = 1` และ `Y0, Y1, Y2, Y3, Y4, Y6, Y7 = 0`
เพราะค่ารหัสฐานสอง `101₂` เท่ากับ `5₁₀`

ถ้าเขียนในรูป minterm จะได้

| เอาต์พุต | เงื่อนไขอินพุต | สมการ active-high |
|---|---|---|
| `Y0` | `ABC = 000` | `EN · A' · B' · C'` |
| `Y1` | `ABC = 001` | `EN · A' · B' · C` |
| `Y2` | `ABC = 010` | `EN · A' · B · C'` |
| `Y3` | `ABC = 011` | `EN · A' · B · C` |
| `Y4` | `ABC = 100` | `EN · A · B' · C'` |
| `Y5` | `ABC = 101` | `EN · A · B' · C` |
| `Y6` | `ABC = 110` | `EN · A · B · C'` |
| `Y7` | `ABC = 111` | `EN · A · B · C` |

แนวคิดสำคัญคือ decoder ไม่ได้ "คำนวณเลข" แต่ "เลือกแถว" ของตารางความจริง
ดังนั้น decoder จึงใช้สร้างฟังก์ชันบูลีนได้โดย OR เฉพาะเอาต์พุตที่เป็น minterm ของฟังก์ชันนั้น

```text
        A ───────────────┐
        B ───────────────┼──> 3-to-8 decoder ── Y0
        C ───────────────┘                      Y1
       EN ────────────────────────────────────> Y2
                                                ...
                                                Y7
```

7-segment display เป็นชุด LED 7 ส่วนชื่อ `a` ถึง `g`
แต่ละส่วนติดหรือดับเพื่อสร้างตัวเลขฐานสิบ `0` ถึง `9`
การระบุ segment ตามมาตรฐานในใบงานนี้ใช้ตำแหน่งต่อไปนี้

```text
        a
     ┌─────┐
   f │     │ b
     │  g  │
     ├─────┤
   e │     │ c
     │     │
     └─────┘
        d
```

ไอซี 7447 เป็น BCD-to-7-segment decoder/driver สำหรับ 7-segment แบบ common-anode
อินพุตเป็นรหัส BCD 4 บิต `D C B A` โดย `A` เป็น LSB
เอาต์พุต `a` ถึง `g` ของ 7447 เป็น active-low หมายความว่า
เอาต์พุตเป็น `0` แล้ว segment นั้นติด และเอาต์พุตเป็น `1` แล้ว segment นั้นดับ

เหตุผลคือใน common-anode display ขาร่วมของ LED ต่อกับ `+5V`
ไอซี 7447 ทำหน้าที่ดึงกระแสลงกราวด์ (sink current)
เมื่อเอาต์พุตเป็น LOW กระแสไหลจาก `+5V` ผ่าน LED ผ่านตัวต้านทาน เข้า output pin ของ 7447 แล้วลงกราวด์

```text
Common anode +5V
        │
        ├── segment a LED ── R ── 7447 output a
        ├── segment b LED ── R ── 7447 output b
        ├── segment c LED ── R ── 7447 output c
        └── ...

7447 output LOW  = segment ON
7447 output HIGH = segment OFF
```

## ตัวอย่างออกแบบก่อนลงมือ

### ตัวอย่างที่ 1: ออกแบบสมการของ 3-to-8 decoder พร้อม enable

ต้องการให้ `ABC = 110` เปิดเฉพาะ `Y6`

1. อ่านค่าบิตทีละตำแหน่ง: `A = 1`, `B = 1`, `C = 0`
2. บิตที่เป็น 1 ใช้ตัวแปรตรง: `A`, `B`
3. บิตที่เป็น 0 ใช้ตัวแปรกลับค่า: `C'`
4. ใส่ enable เป็นเงื่อนไขร่วม: `EN`
5. ได้สมการ `Y6 = EN · A · B · C'`

เมื่อ `EN = 0` ไม่ว่า `ABC` เป็นค่าใด เอาต์พุตทุกเส้นต้องเป็น 0
เมื่อ `EN = 1` และ `ABC = 110` จึงได้ `Y6 = 1`

ตารางตรวจเฉพาะกรณีสำคัญ:

| EN | A | B | C | Y6 ที่คาดหวัง | เหตุผล |
|:-:|:-:|:-:|:-:|:-:|---|
| 0 | 1 | 1 | 0 | 0 | enable ปิด |
| 1 | 1 | 1 | 0 | 1 | ตรง minterm 6 |
| 1 | 1 | 0 | 0 | 0 | `B` ไม่ตรง |
| 1 | 1 | 1 | 1 | 0 | `C` ไม่ตรง |

### ตัวอย่างที่ 2: ใช้ decoder สร้าง `F(A,B,C)=Σm(1,3,5,7)`

ฟังก์ชันนี้มีค่าเป็น 1 ที่ minterm 1, 3, 5 และ 7
ถ้าใช้ decoder 3-to-8 แบบ active-high ให้ต่อเอาต์พุต `Y1`, `Y3`, `Y5`, `Y7` เข้า OR gate 4 อินพุต

```text
                    ┌──────────────┐
 A ────────────────>│              │── Y0
 B ────────────────>│  3-to-8      │── Y1 ──┐
 C ────────────────>│  decoder     │── Y2   │
 EN = 1 ───────────>│              │── Y3 ──┼── OR ── F
                    │              │── Y4   │
                    │              │── Y5 ──┤
                    │              │── Y6   │
                    │              │── Y7 ──┘
                    └──────────────┘
```

ตารางพิสูจน์:

| A | B | C | เลข minterm | เอาต์พุต decoder ที่ active | F |
|:-:|:-:|:-:|---:|---|:-:|
| 0 | 0 | 0 | 0 | `Y0` | 0 |
| 0 | 0 | 1 | 1 | `Y1` | 1 |
| 0 | 1 | 0 | 2 | `Y2` | 0 |
| 0 | 1 | 1 | 3 | `Y3` | 1 |
| 1 | 0 | 0 | 4 | `Y4` | 0 |
| 1 | 0 | 1 | 5 | `Y5` | 1 |
| 1 | 1 | 0 | 6 | `Y6` | 0 |
| 1 | 1 | 1 | 7 | `Y7` | 1 |

จากตารางจะเห็นว่า `F = 1` เมื่อ `C = 1` ทุกกรณี
ดังนั้นฟังก์ชันนี้ลดรูปได้เป็น `F = C`
แต่จุดประสงค์ของ Lab นี้คือให้เห็นว่า decoder ใช้สร้างฟังก์ชันจาก minterm ได้โดยตรง
แม้ยังไม่ลดรูปก็ตาม

### ตัวอย่างที่ 3: หา segment ที่ต้องติดเพื่อแสดงเลข 6 บน 7-segment

เลข 6 ต้องมีรูปร่างดังนี้

```text
        a=ON
     ┌─────┐
 f=ON│     │ b=OFF
     │ g=ON│
     ├─────┤
 e=ON│     │ c=ON
     │     │
     └─────┘
        d=ON
```

ดังนั้นในรูป segment logic แบบ active-high จะได้
`a,b,c,d,e,f,g = 1,0,1,1,1,1,1`

แต่ถ้าใช้ 7447 ซึ่งเป็น active-low output
ค่าที่ขา output ของ 7447 ต้องกลับกันเป็น
`a,b,c,d,e,f,g = 0,1,0,0,0,0,0`

ผู้เรียนต้องแยกให้ชัดว่า "segment ติด" กับ "ขา output ของ 7447 เป็น 1" ไม่ใช่สิ่งเดียวกัน

## ส่วนที่ 1: สร้าง 3-to-8 decoder จากเกตใน Logisim (45 นาที)

### 1.1 ตั้งค่าไฟล์และสัญญาณ

1. เปิด Logisim Evolution
2. สร้างไฟล์ใหม่ชื่อ `lab09-decoder-7segment.circ`
3. ตั้งชื่อวงจรหลักว่า `main`
4. วาง input pin 4 ตัวทางซ้าย และตั้งชื่อเป็น `EN`, `A`, `B`, `C`
5. ตั้งค่า `Data Bits = 1` ให้ input pin ทุกตัว
6. วาง output pin 8 ตัวทางขวา ตั้งชื่อเป็น `Y0`, `Y1`, `Y2`, `Y3`, `Y4`, `Y5`, `Y6`, `Y7`
7. จัดตำแหน่ง output pin เรียงจากบนลงล่าง `Y0` ถึง `Y7`
8. ใช้ Text Tool เขียนกำกับว่า `A = MSB, C = LSB`

### 1.2 สร้างสัญญาณกลับค่า

1. จากหมวด `Gates` วาง NOT gate 3 ตัว
2. ต่อ `A` เข้า NOT ตัวแรก แล้วตั้งชื่อสายออกเป็น `nA`
3. ต่อ `B` เข้า NOT ตัวที่สอง แล้วตั้งชื่อสายออกเป็น `nB`
4. ต่อ `C` เข้า NOT ตัวที่สาม แล้วตั้งชื่อสายออกเป็น `nC`
5. จัด `A/nA`, `B/nB`, `C/nC` ให้อ่านได้เป็นคู่แนวตั้ง

```text
A ──────────────── A
  └─ NOT ───────── nA

B ──────────────── B
  └─ NOT ───────── nB

C ──────────────── C
  └─ NOT ───────── nC
```

### 1.3 วาง AND gate สำหรับ minterm

1. จากหมวด `Gates` วาง AND gate 8 ตัว
2. ตั้งค่า AND gate ทุกตัวเป็น `Number of Inputs = 4`
3. อินพุตแรกของ AND ทุกตัวต่อกับ `EN`
4. อินพุตที่เหลือเลือกต่อ `A/nA`, `B/nB`, `C/nC` ตามตารางนี้

| Output | Input เข้า AND 4 เส้น |
|---|---|
| `Y0` | `EN`, `nA`, `nB`, `nC` |
| `Y1` | `EN`, `nA`, `nB`, `C` |
| `Y2` | `EN`, `nA`, `B`, `nC` |
| `Y3` | `EN`, `nA`, `B`, `C` |
| `Y4` | `EN`, `A`, `nB`, `nC` |
| `Y5` | `EN`, `A`, `nB`, `C` |
| `Y6` | `EN`, `A`, `B`, `nC` |
| `Y7` | `EN`, `A`, `B`, `C` |

5. ต่อ output ของ AND แต่ละตัวไปยัง output pin ที่ตรงกัน
6. ใส่ probe เพิ่มที่ `EN`, `A`, `B`, `C`, `Y0` ถึง `Y7` เพื่อดูค่าระหว่างทดสอบ

แผนผังย่อของ `Y3` และ `Y6`:

```text
Y3 = EN · A' · B · C

EN ─────┐
nA ─────┤
B  ─────┼── AND ── Y3
C  ─────┘

Y6 = EN · A · B · C'

EN ─────┐
A  ─────┤
B  ─────┼── AND ── Y6
nC ─────┘
```

## ส่วนที่ 2: ใช้ decoder และ OR gate สร้างฟังก์ชันบูลีน (30 นาที)

### 2.1 สร้างวงจรย่อย `func_by_decoder`

1. เพิ่มวงจรใหม่ชื่อ `func_by_decoder`
2. คัดลอก decoder จากส่วนที่ 1 หรือสร้างเป็น subcircuit แล้วนำมาใช้
3. วาง input pin `A`, `B`, `C` และต่อ `EN` ของ decoder ให้เป็นค่าคงที่ `1`
4. วาง OR gate 1 ตัว ตั้งค่า `Number of Inputs = 4`
5. ต่อ `Y1`, `Y3`, `Y5`, `Y7` เข้า OR gate
6. ต่อ output ของ OR gate ไปยัง output pin ชื่อ `F`
7. เพิ่ม output pin อีกตัวชื่อ `C_direct` แล้วต่อจาก input `C` โดยตรงเพื่อใช้เทียบผล

```text
Y1 ──┐
Y3 ──┼── OR ── F
Y5 ──┤
Y7 ──┘

C ───────── C_direct
```

### 2.2 ทดสอบฟังก์ชัน `F=Σm(1,3,5,7)`

ให้เปลี่ยนค่า `ABC` ครบ 8 กรณี แล้วเปรียบเทียบ `F` กับ `C_direct`

| A | B | C | Active minterm | F จาก decoder+OR | C_direct | ตรงกันหรือไม่ |
|:-:|:-:|:-:|---:|:---:|:---:|:---:|
| 0 | 0 | 0 | 0 | | | |
| 0 | 0 | 1 | 1 | | | |
| 0 | 1 | 0 | 2 | | | |
| 0 | 1 | 1 | 3 | | | |
| 1 | 0 | 0 | 4 | | | |
| 1 | 0 | 1 | 5 | | | |
| 1 | 1 | 0 | 6 | | | |
| 1 | 1 | 1 | 7 | | | |

### 2.3 วิเคราะห์จำนวนชั้นของลอจิก

ในวงจร decoder จากเกต เอาต์พุตแต่ละ `Y` ผ่าน NOT และ AND
เมื่อใช้ OR รวม minterm จะมีเส้นทางสัญญาณยาวที่สุดประมาณ

```text
Input A/B/C -> NOT -> AND(minterm) -> OR -> F
```

แต่ฟังก์ชันที่ลดรูปแล้วคือ `F = C`
ซึ่งมีเส้นทางสั้นกว่า

```text
Input C -> F
```

ให้ตอบในรายงานว่า เหตุใด decoder จึงยังมีประโยชน์
แม้บางฟังก์ชันสามารถลดรูปได้สั้นกว่า
ให้เชื่อมคำตอบกับกรณีที่มีหลายฟังก์ชันใช้ minterm ร่วมกัน เช่น วงจรควบคุมที่ต้องสร้าง `F1`, `F2`, `F3` จาก input ชุดเดียว

## ส่วนที่ 3: ขับ 7-segment ด้วย Logisim (35 นาที)

### 3.1 สร้างอินพุต BCD และ display

1. เพิ่มวงจรใหม่ชื่อ `bcd_7seg_logisim`
2. วาง input pin 4 ตัว ตั้งชื่อเป็น `D`, `C`, `B`, `A`
3. กำหนดให้ `D` เป็น MSB และ `A` เป็น LSB ของ BCD

## ส่วนที่ 4: ต่อ 7447 กับ 7-segment common-anode บน breadboard (55 นาที)

### 4.1 ขาไอซี 7447 ที่ใช้ในใบงาน

ให้หันรอยบากหรือจุดบนตัวไอซีขึ้นด้านบน แล้วนับขาทวนเข็มนาฬิกา

### 4.2 การต่อ display แบบ common-anode

1. ปิดแหล่งจ่ายก่อนเสียบไอซีทุกครั้ง
2. เสียบ 7447 ข้ามร่องกลางของ breadboard
3. ต่อขา 16 ของ 7447 ไปที่ `+5V`
4. ต่อขา 8 ของ 7447 ไปที่ `GND`
5. ต่อขา `LT`, `BI/RBO`, `RBI` คือขา 3, 4, 5 ไปที่ `+5V`
6. ต่อ common anode ของ 7-segment ไปที่ `+5V`
7. ต่อ output `a` ถึง `g` ของ 7447 ไปยังขา segment ที่ตรงกันผ่านตัวต้านทาน 220 Ω หรือ 330 Ω
8. ต่อสวิตช์ 4 ตัวเป็นอินพุต BCD โดยมี pull-down หรือ pull-up ให้สถานะ 0/1 ชัดเจน
9. กำหนดสวิตช์เป็น `D C B A` จากซ้ายไปขวา เพื่อให้ตรงกับตาราง

แผนภาพการต่ออย่างย่อ:

```text
สวิตช์ BCD
D ───────── pin 6  7447  pin 13 ── R ── segment a
C ───────── pin 2        pin 12 ── R ── segment b
B ───────── pin 1        pin 11 ── R ── segment c
A ───────── pin 7        pin 10 ── R ── segment d
                         pin  9 ── R ── segment e
+5V ─────── pin16        pin 15 ── R ── segment f
GND ─────── pin 8        pin 14 ── R ── segment g

7-segment common anode ─────────────── +5V
LT, BI/RBO, RBI ────────────────────── +5V
```

### 4.3 ตรวจ active-low output ด้วยเลข 0 และเลข 1

ก่อนทดสอบครบทุกเลข ให้ตรวจสองกรณีนี้เพื่อยืนยันว่าประเภท display ถูกต้อง

| เลข | BCD `D C B A` | Segment ที่ควรติด | ขา output ของ 7447 ที่ควรเป็น LOW |

| 0 | 0000 | a, b, c, d, e, f | a, b, c, d, e, f |
| 1 | 0001 | b, c | b, c |

ถ้าใช้ common-cathode display กับ 7447 โดยตรง
ผลมักกลับตรรกะหรือไม่ติดตามที่คาด เพราะ common-cathode ต้องการ driver แบบ active-high/source current
ในใบงานนี้ต้องใช้ common-anode เท่านั้น

## คำถามท้ายใบงาน

1. 3-to-8 decoder กับ demultiplexer 1-to-8 มีโครงสร้างคล้ายกันอย่างไร และต่างกันที่ความหมายของสัญญาณ input ใด
2. ถ้า `EN = 1` และ `ABC = 011` แต่ `Y1` กับ `Y3` ติดพร้อมกัน จงระบุสาเหตุที่เป็นไปได้อย่างน้อย 2 ข้อในวงจรจากเกต
3. จากฟังก์ชัน `F(A,B,C)=Σm(1,3,5,7)` เหตุใดจึงลดรูปเป็น `F=C` ได้ และทำไมในบางระบบยังเลือกใช้ decoder+OR แทนการต่อ `C` ตรง
4. 7447 เป็น active-low output หมายความว่าอย่างไร อธิบายโดยใช้เส้นทางกระแสของ common-anode 7-segment
5. common-anode และ common-cathode 7-segment ต่างกันอย่างไร ถ้านำ common-cathode มาต่อกับ 7447 โดยตรงจะเกิดปัญหาใด
6. อินพุต BCD `1010` ถึง `1111` ไม่ใช่รหัสของเลขฐานสิบ 0-9 หาก display ยังแสดงบางรูปขึ้นมา ควรตีความผลนั้นอย่างไรในการออกแบบระบบดิจิทัล
7. ถ้าต้องการแสดงเลขฐานสิบ 2 หลักด้วย 7-segment สองตัว แต่ต้องการลดจำนวนสาย segment ควรใช้แนวคิด multiplex display อย่างไร และ decoder มีบทบาทตรงไหน
8. ในวงจร decoder จากเกต ถ้า NOT gate ของ `C` มี delay มากกว่าสาย `C` ตรง อาจเกิด glitch ชั่วคราวที่เอาต์พุตใดบ้างเมื่อตัวนับเปลี่ยนจาก `001` ไป `010`
