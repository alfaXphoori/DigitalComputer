Course Description

ระบบตัวเลข เกตเชิงตรรกะ และนิพจน์ตรรกะ พีชคณิตแบบบูล ผังคาร์โนห์ วิธีการใช้ตาราง วงจรตรรกะแบบรวมและการใช้งาน วงจรบวก วงจรลบ วงจรแสดงผลพหุคูณ วงจรถอดรหัส วงจรลงรหัส มัลติเพล็กเซอร์ ดีมัลติเพล็กเซอร์ การใช้งานของเกตไตรสเตด ความเร็วและเวลาของวงจรเชิงตรรก วงจรเชิงลำดับ และการออกแบบ ฟลิปฟลอป เคาน์เตอร์ เรจิสเตอร์

Number System; logic gates and logic expression; Boolean algebra: Karnaugh map, tabulation method; combination logic circuit and applications: adder, subtractor, multiple outputs circuit, decoder, encoder, multiplexer, demultiplexer; gate implementation: tristate, speed and delay in logic circuits; sequential circuits and design; flip-flop, counter; register.

ผลลัพธ์การเรียนรู้ระดับรายวิชา

Course Learning Outcomes (CLOs)

CLO 1

อธิบายพื้นฐานดิจิทัลลอจิก

อธิบายระบบตัวเลข รหัส เกตตรรกะ ตารางความจริง และนิพจน์ตรรกะได้ถูกต้อง

CLO 2

ลดรูปฟังก์ชันบูลีน

ลดรูปฟังก์ชันด้วยพีชคณิตบูลีน ผังคาร์โนห์ และวิธีตาราง พร้อมตรวจสอบผลได้

CLO 3

ออกแบบวงจรเชิงผสม

วิเคราะห์และออกแบบ adder, subtractor, decoder, encoder, mux/demux และวงจรแสดงผลได้

CLO 4

สร้างวงจรเชิงลำดับ

ออกแบบ flip-flop, counter, register และ FSM เบื้องต้น พร้อมจำลอง/ต่อวงจรจริงและทำงานเป็นทีมได้
