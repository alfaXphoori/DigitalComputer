Digital Computer Logic — เรียนรู้ระบบดิจิทัล ระบบจำนวน พีชคณิตบูลีน และการออกแบบ วงจรคอมบิเนชัน/ซีเควนเชียล ควบคู่กับการลงมือต่อและทดสอบวงจรจริงด้วยไอซีและเครื่องมือวัด

Course Description

ระบบตัวเลข เกตเชิงตรรกะ และนิพจน์ตรรกะ พีชคณิตแบบบูล ผังคาร์โนห์ วิธีการใช้ตาราง วจรตรรกะแบบรวมและการใช้งาน วงจรบวก วงจรลบ วงจรแสดงผลพหุคูณ วงจรถอดรหัส วงจรลงรหัส มัลติเพล็กเซอร์ ดีมัลติเพล็กเซอร์ การใช้งานของเกตไตรสเตด ความเร็วและเวลาของวงจรเชิงตรรก วงจรเชิงลําดับ และการออกแบบ ฟลิปฟลอป เคาเตอร์ เรจิสเตอร์

Number System; logic gates and logic expression; Boolean algebra: Karnaugh map, tabulation method; combination logic circuit and applications: adder, subtractor, multiple outputs circuit, decoder, encoder, multiplexer, demultiplexer; gate implementation: tristate, speed and delay in logic circuits; sequential circuits and design; flip-flop, counter; register.

ผลลัพธ์การเรียนรู้ระดับรายวิชา

Course Learning Outcomes (CLOs) — เมื่อจบรายวิชาผู้เรียนจะสามารถ

CLO 1

อธิบายหลักการระบบดิจิทัล

เข้าใจระบบดิจิทัล ระบบจำนวน รหัสดิจิทัล และคุณสมบัติของดิจิทัลไอซี

CLO 2

วิเคราะห์และลดรูปสมการลอจิก

ใช้พีชคณิตบูลีนและแผนผังคาร์โนห์ลดรูปสมการได้อย่างถูกต้อง

CLO 3

ออกแบบวงจรดิจิทัล

ออกแบบและสร้างวงจรคอมบิเนชันและวงจรซีเควนเชียลตามเงื่อนไขที่กำหนด

CLO 4

ปฏิบัติการต่อและทดสอบวงจร

ต่อ ทดสอบ และวิเคราะห์วงจรด้วยไอซีและเครื่องมือวัด พร้อมทำงานเป็นทีม
